## Web GUI

An optional HTML/JS dashboard is available to configure and launch the end-to-end
pipeline.

### Prerequisites

```bash
pip install flask
```

### Usage

```bash
python3 pipeline_server.py
```

Then open [http://localhost:5050](http://localhost:5050) in your browser.  
You can fill in the simulation parameters, start the pipeline, watch the four
stages progress bar, and expand the terminal panel to follow the live logs.
